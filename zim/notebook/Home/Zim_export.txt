====== Zim export ======
Created Monday 12 November 2012

The source of our documentation site (http://doc.webenq.org/) is a Zim notebook (http://www.zim-wiki.org/)
This notebook is stored in github branch 'gh-pages' of https://github.com/Nivocer/webenq/ in zim/notebook
To export the zim notebook we use the template in zim/template/webenq.html
The export can be beformed by executing the shell script 'zim-export' in the root of the gh-pages branch
The html export is stored at the root of this branch
