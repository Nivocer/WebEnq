==== Form: Styling single checkbox or multiple submit buttons ====
purpose: display multiple submitbuttons next to each other or a single checkbox on a single line:
$this->addDisplayGroup(
            array('previous','next','done'),
            'submitTable',
            array('class' => 'table')
        );
in global.css this is styled using fieldset.table
