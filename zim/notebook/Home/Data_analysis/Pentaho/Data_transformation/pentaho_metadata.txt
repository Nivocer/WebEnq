=== Data connection ===
* use jndi data connection, advantages:
	* don't save password in model
	* easy use other database credentials on different servers
* jndi name is name of connection in bi-server / pac
* define jndi connection for use in  pentaho metadata editor at : [studio:PME install directory]/simple-jndi/jdbc.properties, eg
'''
webenq4/type=javax.sql.DataSource
webenq4/driver=org.gjt.mm.mysql.Driver
webenq4/url=jdbc:mysql://localhost:3306/databaseName
webenq4/user=webenq4
webenq4/password=password
'''

 where webenq4 is jndi name of the connection
