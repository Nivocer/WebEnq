====== Config ======

===== BI server =====

==== Session lifetime ====
default session lifetime is 30 minutes, to adjust:
 $BISERVER/tomcat/webapps/pentaho/WEB-INF/web.xml
Look for the setting:
<!-- insert additional servlet mappings -->
<session-config>
<session-timeout>30</session-timeout>
</session-config>

The number is the session-timeout in minutes.


==== Start up ====
Java7 is a bit more strict about custom sorts, this gives error with saiku answer label sorting, hack:
in startPentaho.sh 
add  -Djava.util.Arrays.useLegacyMergeSort=true 
to export CATALINA_OPTS

==== Enable publishing of content ====
You need to set a publisher password in order to allow upload of content:
* go to pentaho_solutions/system/
* edit publisher_config.xml

==== Mondrian settings ====
* in pentaho-solutions/system/mondrian/mondrian.properties:

'''
# Allow the use of aggregates
mondrian.rolap.aggregates.Use=true
mondrian.rolap.aggregates.Read=true
'''


==== Logging Mondrian OLAP queries ====
To enable logging of sql & mdx queries, uncomment the mondrian logging entry (''<appender name="MONDRIAN" ...'') in tomcat/webapps/pentaho/WEB-INF/classes/log4j.xml

===== Administration console =====

==== Admin password ====
The default admin password should be changed:
* Generate a new password hash with
	''java -cp lib/jetty-6.1.2.jar:lib/jetty-util-6.1.9.jar org.mortbay.jetty.security.Password admin <password>''
* Put the OBF string and username in resource/config/login.properties:
	''admin: OBF:1v2j1uum1xtv1zej1zer1xtn1uvk1v1v,admin''
