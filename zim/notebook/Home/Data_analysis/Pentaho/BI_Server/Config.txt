====== Config ======

===== BI server =====
* Admin interface to eg. refresh the plugin adapters http://localhost:8080/pentaho/Admin

==== Session lifetime ====
The default session lifetime is 30 minutes. To adjust, edit $BISERVER/tomcat/webapps/pentaho/WEB-INF/web.xml and change the setting:
'''
<!-- insert additional servlet mappings -->
<session-config>
<session-timeout>30</session-timeout>
</session-config>
'''


==== Don't show user names in login window ====
To remove the drop-down of evaluation logins, edit pentaho-solutions/system/pentaho.xml and set:
'''
<login-show-users-list>false</login-show-users-list>
<login-show-sample-users-hint>false</login-show-sample-users-hint>
'''


==== Enable publishing of content ====
You need to set a publisher password in order to allow upload of content:
* go to pentaho_solutions/system/
* edit publisher_config.xml

==== Mondrian settings ====
* in pentaho-solutions/system/mondrian/mondrian.properties:

'''
# Allow the use of aggregates
mondrian.rolap.aggregates.Use=true
mondrian.rolap.aggregates.Read=true
'''


==== Logging Mondrian OLAP queries ====
To enable logging of sql & mdx queries, uncomment the mondrian logging entry (''<appender name="MONDRIAN" ...'') in tomcat/webapps/pentaho/WEB-INF/classes/log4j.xml

==== Mysql Socket error fix ====
http://wiki.pentaho.com/display/ServerDoc2x/Configuring+for+MySQL
* download http://sourceforge.net/projects/c3p0 en save lib/* in tomcat/webapps/pentaho/WEB-INF/lib
* insert in to pentaho-solutions/system/hibernate/mysql5.hibernate.cfg.xml,  after the <session-factory> tag and just before the <!-- MySQL Configuration --> comment.: 
'''
<!-- hibernate c3p0 settings -->
<property name="connection.provider_class">org.hibernate.connection.C3P0ConnectionProvider</property>
<property name="hibernate.c3p0.acquire_increment">3</property>
<property name="hibernate.c3p0.idle_test_period">10</property>
<property name="hibernate.c3p0.min_size">5</property>
<property name="hibernate.c3p0.max_size">75</property>
<property name="hibernate.c3p0.max_statements">0</property>
<property name="hibernate.c3p0.timeout">25200</property>
<property name="hibernate.c3p0.preferredTestQuery">select 1</property>
<property name="hibernate.c3p0.testConnectionOnCheckout">true</property>
<!-- End of patch -->
'''


===== Administration console =====

==== Admin password ====
The default admin password should be changed:
* Generate a new password hash with
	''java -cp lib/jetty-6.1.2.jar:lib/jetty-util-6.1.9.jar org.mortbay.jetty.security.Password admin <password>''
* Put the OBF string and username in resource/config/login.properties:
	''admin: OBF:1v2j1uum1xtv1zej1zer1xtn1uvk1v1v,admin''
