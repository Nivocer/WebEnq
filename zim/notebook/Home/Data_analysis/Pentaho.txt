=== Meta data ===
link between database and user (predefined report, dashboard)
tool: pentaho metadata editor
documentation: http://wiki.pentaho.com/display/COM/The+Pentaho+Metadata+Project

=== Mondrian: ===
link between database and user (olap)
tool to define mondrian schema: pentaho schema workbench 
tool to create aggregation: pentaho aggregate designer (not used yet)
tool to build mdx queries: saiku
documentation: http://mondrian.pentaho.com/documentation

=== Olap/ad hoc analytics ===

http://docs.analytical-labs.com/

=== Predefined reports ===
tool: pentaho report designer
documentation: http://wiki.pentaho.com/display/Reporting/Pentaho+Reporting+Community+Documentation
can use table, jdbc, metadata or mdx as datasource
