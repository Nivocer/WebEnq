Architecture

The core of WebEnq is written as a PHP/Zend application.

Reports are generated with Jasper, which is a Java application.

Questionnaires can be exported to Xforms and then imported into Open Data Kit (ODK) to be deployed on Android-based devices

Installation instructions and detailed requirements are provided in an INSTALL file with the source code.

Hosted on GitHub Pages based on the Dinky theme
